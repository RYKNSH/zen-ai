# /l2 - Autonomous Mode Shortcut

## 使用方法

```bash
/l2 "タスク"
```

**実行内容**:
1. `/level 2` を実行（Autonomous Mode - デフォルト）
2. `/go "タスク"` を実行

**効果**: 自動実行、透明性は維持

---

## 実装

```bash
# Level 2に設定
/level 2

# /go を実行
/go "$1"
```

---

## いつ使う

- 通常のタスク（推奨）
- 効率的に作業したい時
- ほとんどの場合

---

> [!TIP]
> `/l2 "タスク"` = `/level 2` + `/go "タスク"` = `/go "タスク"`（デフォルト）
