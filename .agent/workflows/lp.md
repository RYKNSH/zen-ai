# /lp — LP構成案ワークフロー

> High Conversion LP Architect スキルを使って、成約率に特化したLP構成案を作成する。

## 前提条件

- ユーザーから以下の情報を取得すること:
  - **ターゲット**: どんな人に売りたいか
  - **商材**: 何を売るか（サービスの概要、特徴）

## ワークフロー

### Step 1: ヒアリング

ターゲットと商材が未提供の場合、以下を質問:

```
1. ターゲット（誰に売りたいか）を教えてください
2. 商材（何を売るか）を教えてください
```

### Step 2: スキル実行

`high-conversion-lp-architect` スキルを読み込んで実行する:

```
view_file ~/.gemini/antigravity/skills/high-conversion-lp-architect/SKILL.md
```

スキルの6ステップを順番に実行:

1. **顧客の深層心理特定** — ターゲットの「本音の悩み」を30個抽出
2. **PASTOR法構成案** — 購買心理に基づいた構成案を作成
3. **フック生成** — ファーストビューのキャッチコピー100案
4. **客観的証拠とFAQ** — 信頼性担保コンテンツ
5. **デザイン指示書** — デザイナー向け実装指示書
6. **ABテスト案** — 改善用の別パターン

### Step 3: 成果物

すべての生成結果をまとめたマークダウンファイルを作成:
- `lp_plan_for_{product}.md`

### Step 4: 構造生成（オプション）

ユーザーが実装を望む場合、`/generate-lp-structure` を実行してHTMLコードを生成する。
