# 開発サーバー起動

// turbo-all

1. 依存関係がインストールされているか確認し、なければインストール

```bash
pnpm install
```

2. 開発サーバーを起動

```bash
pnpm dev
```

## 注意事項

- ポートはプロジェクト設定に依存（通常3000, 5173, 5174など）
- Hot Reloadが有効
- 開発モードではソースマップが有効

## トラブルシューティング

ポートが使用中の場合：
```bash
lsof -i :<ポート番号> | grep LISTEN
kill -9 <PID>
```
