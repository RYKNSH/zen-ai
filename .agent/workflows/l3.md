# /l3 - Full Auto Mode Shortcut

## 使用方法

```bash
/l3 "タスク"
```

**実行内容**:
1. `/level 3` を実行（Full Auto Mode）
2. `/go "タスク"` を実行

**効果**: 完全自動実行、確認なし

---

## 実装

```bash
# Level 3に設定
/level 3

# /go を実行
/go "$1"
```

---

## いつ使う

- 完全自動化したい時
- 信頼できるタスク
- 上級者向け

---

> [!WARNING]
> Level 3は確認なしで全て実行します。慎重に使用してください。

> [!TIP]
> `/l3 "タスク"` = `/level 3` + `/go "タスク"`
