# /l0 - Manual Mode Shortcut

## 使用方法

```bash
/l0 "タスク"
```

**実行内容**:
1. `/level 0` を実行（Manual Mode）
2. `/go "タスク"` を実行

**効果**: 全ての操作で確認プロンプトが表示される

---

## 実装

```bash
# Level 0に設定
/level 0

# /go を実行
/go "$1"
```

---

## いつ使う

- 初めてのタスク
- 慎重に作業したい時
- 学習目的

---

> [!TIP]
> `/l0 "タスク"` = `/level 0` + `/go "タスク"`
