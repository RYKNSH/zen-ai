# /l1 - Supervised Mode Shortcut

## 使用方法

```bash
/l1 "タスク"
```

**実行内容**:
1. `/level 1` を実行（Supervised Mode）
2. `/go "タスク"` を実行

**効果**: 重要な操作のみ確認プロンプトが表示される

---

## 実装

```bash
# Level 1に設定
/level 1

# /go を実行
/go "$1"
```

---

## いつ使う

- 重要なタスク
- 確認しながら作業したい時
- バグ修正

---

> [!TIP]
> `/l1 "タスク"` = `/level 1` + `/go "タスク"`
